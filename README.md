Understanding MPEG Boxes/Atoms with Node.JS
===

A sample code repo to help understand the structure of MPEG video containers.

### Step 1 - install
>$ npm install

### Step 2 - run
>$ npm run start

### Step 3 - play
>$ npm run play
Play with the code yourself!
